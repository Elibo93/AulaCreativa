### Entorno de desarrollo
---

Este documento describe los entornos de desarrollo, ejecución y herramientas utilizadas para implementar el proyecto **AulaCreativa**, una aplicación desarrollada en Java siguiendo una arquitectura por capas.

#### Lenguaje y versión  
- **Java 17** como lenguaje principal del proyecto

#### Frameworks y librerías principales 
- **Spring Boot** como Framework de la apllicación.
- **Thymeleaf** como motor de plantillas.
- **Spring Data** como capa de abstracción de acceso a datos.
- **Spring Data JPA** como capa de persistencia basada en JPA.
- **JPA** como API de mapeo objeto-relacional(ORM).
- **Hibernate** como implementación ORM.
- **Base de datos H2** como motor de datos en memoria.
- **Maven** como gestor de dependencias.
- **JUnit 5** como framework de pruebas automatizadas.
