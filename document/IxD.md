### IxD (Interacción)
---
[Volver](/README.md)La interacción entre el usuario y el sistema es bastante sencilla y fácil de entender. Se utilizan botones claros para las acciones más comunes, como editar o eliminar. Los formularios tienen los campos bien indicados, así que sabes qué tienes que rellenar en cada momento.

Además, antes de borrar un alumno, el sistema pide confirmación para evitar errores. Después de realizar cualquier acción, se muestra una respuesta inmediata, como volver al listado actualizado o redirigir a otra pantalla.

Todo esto ayuda a que el usuario cometa menos fallos y se sienta más seguro al usar la aplicación.
